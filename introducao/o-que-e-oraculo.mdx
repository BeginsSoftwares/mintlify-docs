## Mercados de Predição

Mercados de predição são plataformas onde as pessoas negociam contratos sobre o resultado de eventos futuros. O preço de cada contrato reflete, de forma coletiva, a **probabilidade estimada** de aquele evento acontecer.

Por exemplo, se um mercado pergunta *"O Brasil vai ganhar a Copa de 2026?"* e os tokens SIM estão sendo negociados a R$ 0,65, isso indica que o mercado coletivamente estima uma probabilidade de 65% de o Brasil ganhar.

## O que é o Oraculo

O Oraculo é uma plataforma de mercados de predição construída sobre a **blockchain Solana**, onde os usuários podem:

- **Criar mercados** sobre qualquer evento com desfecho binário (SIM/NÃO)
- **Negociar posições** comprando ou vendendo tokens de resultado
- **Receber payouts** em USDC quando o mercado é resolvido corretamente

Diferente de cassinos ou apostas esportivas tradicionais, o Oraculo utiliza um **livro de ordens (orderbook)** para precificação — ou seja, os preços emergem da oferta e demanda real entre os participantes, não de uma casa de apostas.

## Diferença para apostas tradicionais

| | Apostas Tradicionais | Oraculo |
|---|---|---|
| **Precificação** | Casa define as odds | Mercado define os preços |
| **Custódia** | Plataforma retém os fundos | Você controla sua carteira |
| **Transparência** | Opaco | On-chain e auditável |
| **Liquidez** | Limitada pela casa | Gerada pelos próprios usuários |
| **Moeda** | BRL (fiat) | USDC (stablecoin) |

## Quem pode usar

O Oraculo é aberto para qualquer pessoa com acesso a internet e uma conta verificada. Para operar, você precisa:

1. Criar uma conta (e-mail, Google ou GitHub)
2. Depositar USDC via PIX (conversão automática de BRL)
3. Escolher um mercado e tomar sua posição

<Note>
  O Oraculo processa depósitos exclusivamente via **PIX**, convertendo reais em USDC automaticamente. Não é necessário ter cripto previamente.
</Note>
